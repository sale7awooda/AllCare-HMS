# AllCare HMS - Development Roadmap

## Phase 1: Foundation (Completed ✅)
*   ✅ Project scaffolding (React + Node + SQLite).
*   ✅ Secure JWT Authentication.
*   ✅ Role-Based Access Control (RBAC) System.
*   ✅ Dynamic Theming & Localization Engine.

## Phase 2: Core Clinical Modules (Completed ✅)
*   ✅ **Patient Registry:** Full CRUD, Search, and 360° Profile.
*   ✅ **Appointments:** Kanban Queue, Daily Token logic, Conflict prevention.
*   ✅ **Billing:** Invoice generation, Partial Payments, Refunds, Insurance handling.
*   ✅ **Admissions:** Interactive Bed Management, Ward View, Discharge Logic.

## Phase 3: Specialized Services (Completed ✅)
*   ✅ **Laboratory:** Test Catalog, Request Workflow, Result Entry (Composite Tests), Reporting.
*   ✅ **Operations:** Surgical Cost Estimator, Team Assignment, Consumables tracking.
*   ✅ **HR & Payroll:** Attendance tracking, Leave requests, Auto-Payroll generation with delta updates.
*   ✅ **Treasury:** Income/Expense tracking, Payment Method analysis.

## Phase 4: System Intelligence & Operations (Completed ✅)
*   ✅ **Analytics:** Financial Revenue Trends, Operational Stats, Demographics.
*   ✅ **Audit Log:** Unified "Records" system tracking all entity creations/updates.
*   ✅ **Data Tools:** Database Snapshot (Backup), Restoration, and Factory Reset.
*   ✅ **Health Diagnostics:** Server and Database performance monitoring.

## Phase 5: Future Enhancements (Planned)
*   **Pharmacy Module:** Dedicated inventory management, batch tracking, and prescription dispensing integration.
*   **Radiology PACS:** Basic integration for uploading and viewing X-Ray/CT images.
*   **Telemedicine:** Integrated video calling for remote outpatient consultations.
*   **Patient Portal:** Separate frontend for patients to view results and book appointments.
*   **SMS Gateway:** Automated SMS notifications for appointment reminders (currently simulated).
